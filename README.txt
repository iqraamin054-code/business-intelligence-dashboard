Business Intelligence Analytics Dashboard

Description:
A responsive Business Intelligence Analytics Dashboard developed for the TEYZIX CORE Frontend Web Development Internship.

Features:

* KPI Dashboard
* Analytics Charts
* Customer Reports
* Search, Sorting and Pagination
* Light/Dark Theme
* Redux Toolkit State Management
* API Integration using Axios
* CSV Export
* PDF Export
* Responsive Design

Technologies Used:
React.js
JavaScript
Redux Toolkit
Axios
Recharts
React Router
CSS
Vite

Run Instructions:

1. Open the project folder.
2. Run:
   npm install
3. Run:
   npm run dev
